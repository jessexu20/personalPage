Past Projects:

  1. DireWolf Job Portal App (Academic Project)

    Ruby on Rails

    During this project, we have built a job hunting website which will help job seekers to look for jobs and employers to find suitable candidates for open positions.

    • Briefly,the website has the following function, more details can be viewed at Github Link :
      • Searching can be performed case insensitive on the all job fields (including title, description, field, Employer ,Tags).
      • Accounts can be deleted and modified by the Account holder and Admins.
      • Employer account can only be created by Admins. An employer can create/edit/delete more than one jobs that they have posted, add tags to them, see the JobApplications on the Job(s) that they have created,update the status of the job applications on their posted jobs.
      • Job Seeker can view and apply to a job, withdraw an existing application but cannot apply to an out-dated job.
  2. Re-factoring grades_controller in Expertiza(Academic Project, Open-Source)

    Ruby on Rails,JavaScript

    3. During the Open-Source project, we have greatly improved the performance of Expertiza System, an Education Website used by North Carolina State University. We followed the RESTful design and Object-Oriented Design Principles to re-factor the methods in grades_controller and implemented the mailing features in the peer-reviewing system.

    • We have done the following modifications to the website. More details can be found on our Wiki and Github Link.
      • We eliminated unnecessary if and unless loops in calculate_penalities method and made it stick to the Ruby Conventions and features by shortening the code into 32 lines from 45 lines without changing its function.
      • Used Delegation Pattern for mailing features, thus maintained the MVC framework of the functionality.
      • Changed the page rendering sequence so that detailed information won't be rendered until needed which improved the page rendering speed by 98% from 484988ms to 8642ms.
  3. Handover Technologies among Heterogeneous Networks on SmartPhones (Research Project)

    Java,Matlab

    4. During this project, I finished an Android Application Project, cooperating with Shanghai Jiao Tong University Wu Xi Research Centre. The mobile application will automatically do the network handover for customers according to the network features and user moving pattern and offer a seamless network handover experience to users. I also put forward a patent on this heterogeneous network handover algorithm.

    • The application has the following features. The patent details can be found at here.
      • Help you choose a network interface (WiFi/Cellular Network) based on QoS requirements of your current running applications. (Many more attributes other than RSSI)
      • Automatically decide when to make a handover so that the handover process will be least noticed by the user based on user's moving pattern.
      • the influence of Load balance of the Wireless AP is concerned in simulations.
  4. High Intelligent Geo-technical Sensor Signal Processing(Research Project)

    C#

    5. During this project, we developed a desktop software to monitor the cables status of suspension bridges.

    • The application has the following features.
      • The software is able to give alert to bridge engineers if detected something unusual to the bridge cables.
